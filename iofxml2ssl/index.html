IOF XML results to SSL converter

This tool is for converting IOF XML results to SSL (Irma) format described here. Mainly intended for converting results from EResults Lite (by Oriento) that does not support the required result format natively.

Using this tool is free but user also remains responsible of the results after the conversion. For errors and improvements please create an issue at github. NOTE: Result prosessing is done entirely in the browser so results are not sent or stored anywhere.

If your IOF XML does not have IRMA license numbers included, add the needed information here before generating results.

IRMA competitor database can be found here: IRMA-database.

Licenses cannot be read directly from IRMA by this application so open the above link to a new tab and Copy/Paste the information to below field and press Read licenses button. After that you can generate the IRMA results and the license numbers will be added to the results automatically based on competitor emit number if not available on IOF XML.

If you need to add or replace some information, add it to the end of the input field using similar IRMA format: 

  [license-number];[first name];[last name];[gender];[year of birth];[club];[something];[something];[emit-number]
  e.g.
  111307;;;;;;;;506336

It will add or replace existing info that might be available earlier on the input data.
